Class dom.xhr

Methods

get

Object get ( url , success , error )
Method / Shortcut for making Ajax Get Requestes
Parameters:
url <String> The Url to which the Ajax call should be made
success <Function> The Success Callback will be called if the Request was made Successful (readyState === 4, httpStatus === 200).
error <Function> The Error Callback will be called if the Server responded with a different StatusCode than 200
Returns: Object
Returns the XMLHttpRequest Object for that Request.
