private Function getIE7Constructor ( )
Returns a Constructor Function which basically is an Array Object borrowed from an iframe. IE 7 and below can't handle a Constructor Function whichs prototype is set to Array.prototype, so we create an iframe and pass the Array Object from the iframe to its parent Window. We then use this regular Array Object and extending it's prototype.
