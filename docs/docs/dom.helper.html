Methods

loadScript

Boolean loadScript ( url , id , callback )
Method for loading Javascript Files asynchronously
Parameters:
url <String> the Url of the Script that should be loaded
id <String> A unique Name to prevent loading the same file twice (this ID will be used as a Dom ID)
callback <Function> Optional: A Callback Function which should be executed when the Script is fully loaded (the callback gets also executed if the file is already loaded)
Returns: Boolean
Returns false if the Script is already loaded

once

Function once ( fn )
Method that takes a function and returns a function which can only be executed once
Parameters:
fn <Function> The Function you want to 'protect' for executing more than once
Returns: Function
Returns a Function which can only be executed once (deletes the reference to the original function and fails silently)
