setSelectorEngine

void setSelectorEngine ( engine )
If the Browser doesn't support querySelectorAll add the Method setSelectorEngine to domtools which replaces the noop function selectorEngine with i.e. Sizzle or something else. This Method will only be avaialable if the Browser doesn't support querySelectorAll, so you have to test if this method exists and if it's true you must call it and pass a CSS Selector Engine. Otherwise domtools.query and domtools.byClass will throw an Error everytime you call them. The easiest way: Create an init function for your application and check if domtools.setSelectorEngine exists, if not you could call your init function, otherwise load Sizzle and call then your init function.
Parameters:
engine <Function> The Selector Engine you want to use
Returns: void
